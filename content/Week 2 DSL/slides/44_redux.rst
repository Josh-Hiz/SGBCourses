Tracing a while loop, redux
===========================

.. quizdown::

    ## Consider this snippet of code:

    ```python
    num = 1
    while num < 0:
        num += 1
    ```

    Trace through the code. How many times is line A executed and how many times is line B executed

    1. [x] A: 1, B: 0
    2. [ ] A: 2, B: 1
    3. [ ] A: 1, B: 1
    4. [ ] A: 0, B: 0
