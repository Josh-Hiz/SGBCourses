    ## So far we've seen three different datatypes. What are they?

    > Datatypes are the *types* that *data* can have.<br>Variables(A) are where we store data.<br>`and`(B) is an operator on booleans.<br>If statements (F) are a language construct for choosing control flow based on booleans.

    - [ ] variables
    - [x] strings
    - [x] booleans
    - [ ] and
    - [x] numbers (ints, floats)
    - [ ] if statements
