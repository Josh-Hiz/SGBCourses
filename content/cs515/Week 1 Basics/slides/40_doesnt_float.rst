It Dosen't Float!
=================

.. quizdown::

    ## Which of the following code segments will evaluate to an *integer*?

    > Manually converting floats yields an int (A); operations other than / on ints yield ints (C). <br> (B) and (D) promote to floats, while (E) converts. 

    - [x] ```int(5 * 5.0)```
    - [ ] ```5 * 2.5```
    - [x] ```5 + 2```
    - [ ] ```5 + 2.0```
    - [ ] ```float(5 * 2)```
