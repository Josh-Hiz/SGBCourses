    ## Which of the following code segments will evaluate to a *floating-point number*?
    (Hint: try to predict it yourself, but you can use a Python interpreter to test.)


    - [x] ```5 + 5.0 // 10```
    - [x] ```11 * 11 // 10.0```
    - [ ] ```10 + 2 // 3```
    - [x] ```5 / 10 // 11```
