Validating Values
=================

.. quizdown::

    ## Given the following if-statement below, which values of ```x``` will result in "The value of x is valid" being printed?

    ```python
    if x > 100 and x < 200:
        print("The value of x is valid")
    ```

    - [x] 139
    - [x] 111
    - [ ] 50
    - [ ] 200
    - [ ] 100
    - [ ] 203
