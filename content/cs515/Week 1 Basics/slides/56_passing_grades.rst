Passing Grades
==============

Write a function ``is_passing`` that takes in ``grade_percentage`` as an input parameter. If grade_percentage is greater than or equal to 60, then the function should return ``True``, otherwise it should return ``False``.

**Sample Input 1:**

``85``

**Sample Output 1:**

``True``

**Sample Input 2:**

``25``

**Sample Output 2:**

``False``

**Sample Input 3:**

``1500``

**Sample Output 3:**

``True``

**Sample Input 4:**

``-300``

**Sample Output 4:**

``False``

**Sample Input 5:**

``60``

**Sample Output 5:**

``True``

**Sample Input 6:**

``59.99``

**Sample Output 6:**

``False``
