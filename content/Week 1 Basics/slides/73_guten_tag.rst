    # Question

    What will be printed when the following code is run?

    ```python
    def print_two(string1, string2):
        print(string1 + " " + string2)
        return

    string1 = "Buenos"
    string2 = "Dias"
    print_two(string2, string1)
    ```

    >>>Notice that ```string2``` is passed as the first actual argument, and will be the first parameter (formal argument)---confusingly, named ```string1```.
