Passing Grades
==============

Write a function ``is_passing`` that takes in ``grade_percentage`` as an input parameter. If grade_percentage is greater than or equal to 60, then the function should return ``True``, otherwise it should return ``False``.

**Sample Input 1:**

.. code-block::

   85

**Sample Output 1:**

.. code-block::

   True

**Sample Input 2:**

.. code-block::

   25

**Sample Output 2:**

.. code-block::

   False

**Sample Input 3:**

.. code-block::

   1500

**Sample Output 3:**

.. code-block::

   True

**Sample Input 4:**

.. code-block::

   -300

**Sample Output 4:**

.. code-block::

   False

**Sample Input 5:**

.. code-block::
   
   60

**Sample Output 5:**

.. code-block::

   True

**Sample Input 6:**

.. code-block::

   59.99

**Sample Output 6:**

.. code-block::

   False
