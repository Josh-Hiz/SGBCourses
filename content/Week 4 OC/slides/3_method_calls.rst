Which are method calls?
=======================

.. quizdown:: 

    # Select every example which is a method call.

    - [ ] ```math.sqrt(16)```
    - [x] ```[1,2,3].pop()```
        > Functions in modules are not methods (so A is not a method call); ```args``` is a field, not a method (so D is not a method call).
    - [x] ```"-".join([867,5309])```
        > Functions in modules are not methods (so A is not a method call); ```args``` is a field, not a method (so D is not a method call).
    - [ ] ```ValueError('wotcher').args```
