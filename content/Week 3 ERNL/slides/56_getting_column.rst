Getting a column
================

.. quizdown::
        
    ### The following 3x4 image is given as a 2D list ```colorful```.

    ![](https://i.ibb.co/8NBNPLP/colorful.png)
    <br>
    What code will print the RGB information in the following image:<br>
    ![](https://i.ibb.co/PN7Hk2q/colorful-2cols.png)

    1. [x] ```
        for row in range(3):
            for col in range(2,4):
                print(colorful[row][col])
        ```
    2. [ ] ```
        for row in range(3):
            for col in range(3):
                print(colorful[row][col])
        ```
    3. [ ] ```
        for row in range(3):
            for col in range(4):
                print(colorful[row][col])
        ```
    4. [ ] ```
        for row in range(1):
            for col in range(4):
                print(colorful[row][col])
        ```
